# Installation

```sh
$ npm install sinopia
$ npm install sinopia-mongodb
```

## Config

Add to your `config.yaml` (example config):

```yaml
auth:
  mongodb:
    host: localhost
    port: 27017
    login: ""
    password: ""
    database: "sinopia"
```
